este circuito nos muestra principalmente un motor o fuente de energia que es la bateria, tambien hay un interruptor que es lo que corta y deja fluir la corriente electrica para que pase luz y poder hacer funcionar al circuito electrico, tambien tenemos las lamparas o bombillos.
la bateria funciona de fuente de energia para hacer que las bombillas se enciendan, el interruptor hace que el circuito se una y se separe cuando se separa, las bombillas permanecen apagadas ya que no hay una corriente constante que la mantenga encendida, mientras que cuando el interruptor une el circuito las bombillas se encienden porque hay una fuente electrica constante.
